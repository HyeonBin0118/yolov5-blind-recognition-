인공지능 기반 시각장애인 보행 안내 시스템

YOLOv5 기반 객체 탐지를 활용하여 시각장애인의 특징(흰 지팡이, 안내견 하네스 등)을 실시간으로 인식하고, 이를 통해 횡단보도 음성 안내 시스템을 자동 활성화하는 연구 프로젝트입니다.

프로젝트 개요

기존 시각장애인용 음성 신호 시스템은 버튼 위치 탐색의 어려움, 고장 방치 문제 등 여러 한계를 가지고 있습니다.
본 연구는 AI 기반 객체 탐지를 통해 시각장애인을 자동 검출하고, 별도의 버튼 조작 없이 보행 보조 기능을 활성화하는 것을 목표로 합니다.
